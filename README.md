# translate-loader

A Webpack Loader For Translate.

## Features

* Compatible with Webpack

## Install

```bash
npm i translate-loader -D
```

## Usage

### Vue usage

```
{
    test: /\.(js|vue)$/,
    loader: 'language-tw-loader',
}
```


### React usage

```
{
    test: /\.(js|jsx)$/,
    loader: 'language-tw-loader',
}
```
